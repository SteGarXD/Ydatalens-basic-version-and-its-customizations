# Описание добавленных функций

## Реализованные функции

### 1. Модуль администрирования пользователей и групп
- CRUD операции для пользователей
- Управление группами
- Интеграция с системой аутентификации DataLens

### 2. RBAC (Разделение доступа)
- Роли: Admin, Analyst, Viewer, Flight Manager
- Права доступа к дашбордам и датасетам
- Проверка прав на уровне API

### 3. LDAP/Active Directory авторизация
- Интеграция с корпоративным AD
- Синхронизация пользователей
- Настройка через UI

### 4. Кросс-фильтрация виджетов
- Взаимное влияние виджетов на дашборде
- Настройка связей через UI
- Фильтрация через URL параметры

### 5. Экспорт дашбордов
- PDF экспорт с русскими шрифтами
- PPT экспорт (редактируемые слайды)
- Markdown экспорт

### 6. API для автоматизации
- REST API для создания/управления чартами
- Программное управление дашбордами
- JWT аутентификация

### 7. Secure Embedding
- Встраивание в защищенные порталы
- JWT токены для безопасности
- Проверка прав доступа

### 8. Версионирование дашбордов
- Автоматическое сохранение версий
- История изменений
- Откат к предыдущим версиям

### 9. Автосохранение
- Автоматическое сохранение каждые N секунд
- Индикатор статуса
- Восстановление несохраненных изменений

### 10. Кастомные виджеты
- Интеграция с Yandex DataSphere
- Загрузка кастомных виджетов
- Sandbox для выполнения скриптов

### 11. Расширенные визуализации
- Drill-Down/Up/Through
- Параметризация чартов
- Parent-Child визуализация
- Combo-чарты

### 12. Дополнительные коннекторы
- Google Sheets
- Yandex Metrica
- AppMetrica

### 13. Материализация данных
- Режим предварительной загрузки в ClickHouse
- Периодическое обновление
- Оптимизация запросов

### 14. Загрузка файлов
- CSV/Excel upload
- Парсинг и загрузка в ClickHouse
- Интеграция с датасетами

### 15. UI/UX улучшения
- Каталоги с личными папками
- Полнотекстовый поиск
- Комментирование
- Мультиязычность (RU/EN)
- Адаптивная верстка

### 16. Брендинг
- Замена логотипа на OP-compass.png
- Текст "AeronavigatorBI"
- Цветовая схема ООО "Аэронавигатор"
- Глубокая CSS кастомизация
- Встраивание в корпоративный стиль

### 17. Специфичные функции для авиации
- Агрегации по рейсам
- Анализ маршрутов
- Расчет ЗПК %
- Интеграция с Meridian
